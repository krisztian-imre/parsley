start_datetime:
• If you can't find distinct start date for and event, use publish_datetime: {1973-01-31 11:11:11} for start_datetime with (MM:00, SS:00)

end_datetime:
• If you can't find end date for an event, use publish_datetime for end_datetime with (MM:30, SS:00)    

token_andor_pair:
• Affected token (XYZ) or trading pair (XYZ/BCD, XYZ_BCD) 
• Use slash '/' instead of underscore '_' for trading pairs.

event_summary:
• Less than 165 characters 
• Focus on key information and indicate if user intervention (e.g., adjusting positions) is necessary.
• Use slash '/' instead of underscore '_' for trading pairs.
